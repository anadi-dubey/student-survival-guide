Student Survival Guide

An open-source financial dashboard designed to help students survive the semester without going broke.

Features
- Daily Safe Spend: Tells you exactly how much you can spend today.
- Ramen Calculator: Translates currency into packs of instant noodles.
- Visualizations: Burn-down chart for your semester budget.

Installation
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/student-survival-guide.git`
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
